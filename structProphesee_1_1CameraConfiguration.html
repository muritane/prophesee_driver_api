◆ max_events_lifespan_us

uint32_t Prophesee::CameraConfiguration::max_events_lifespan_us = 0

Events lifespan.

Only the events that occurred in the last max_events_lifespan_us microseconds are processed. Events occurred earlier are ignored. Updated with Camera::set_max_events_lifespan.
If 0, then all events are processed.

◆ serial_number

std::string Prophesee::CameraConfiguration::serial_number = ""

Serial number of the camera.

In case of an online source, it is the serial number of the current online source.
In case of an offline source, it is the serial number of the online source used to record the data.
