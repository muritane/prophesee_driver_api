Detailed Description

template<typename EventType>
class DATFileWriter< EventType >

Class that will write a DAT file. It is templated on an enum that describes the type of events stored in the DAT file. This enum is used in a trait class that lists the properties of the event (size, name, etc.)

Examples:
prophesee_raw_to_dat.cpp.
