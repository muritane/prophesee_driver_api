◆ set_contrast_sensitivity()

void Prophesee::Biases::set_contrast_sensitivity ( uint32_t  value)

Sets the sensor's sensitivity to the contrast change (both positive and negative changes)

The higher is the sensitivity, the lower is the change of contrast required to trigger a CD/EM event, and vice versa. Increasing the value increases the event rate and the noise, and vice versa.

Parameters
valuethe value to set (in the range from 0 to 100). Input value is clamped in this range.
Warning
Modifying this value may affect the contrast polarity sensitivity (set_contrast_sensitivity_to_polarity)

◆ set_contrast_sensitivity_to_polarity()

void Prophesee::Biases::set_contrast_sensitivity_to_polarity ( uint32_t  value)

Sets the sensor's sensitivity trade off between positive and negative (polarity) contrast change.

The higher the value, the higher is the sensitivity to a positive change of contrast, and vice versa.

Parameters
valuethe value to set (in the range from 0 to 100). Input value is clamped in this range.
Warning
Modifying this value may affect the contrast sensitivity (set_contrast_sensitivity)
