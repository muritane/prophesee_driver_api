Prophesee Driver API Documentation

This API documentation describes Prophesee Driver API that you can use to access data from an event-based Prophesee camera, visualize the data, and control the camera settings.

Main modules

  • Data (events) access and camera control with Prophesee::Camera

Code samples

Code samples described in Examples are installed in /usr/share/prophesee_driver/samples/.

To compile a code sample, first make its copy, go to the directory with your copy and compile it.

mkdir build
cd build
cmake ..
make
