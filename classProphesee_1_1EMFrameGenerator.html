bool Prophesee::EMFrameGenerator::start ( std::uint16_t  fps,
const std::function< void(const Prophesee::timestamp &, const cv::Mat &)> &  cb 
)

Starts the generator thread.

Parameters
fpsFrame rate
cbFunction to call every time a new frame is available. It takes in input the time (in us) of the new frame and the frame. The frame passed as a parameter is guaranteed to be available and left untouched until the next time the callback is called. This means that you don't need to make a deep copy of it, if you only intend to use the frame until the next one is made available by the callback.
Returns
True if the thread started successfully, false otherwise. Also returns false, if the thread is already started.

em_frame_generator.cpp, and prophesee_viewer.cpp.
